## Compilation
To compile the EnderVaults plugin, you need to have the following tools:
- Java JDK 1.8 or higher
- Maven

Run the following command to compile:
```bash
mvn clean package
```

## Installation
1. Download the EnderVaults 1.2.0 jar file.
2. Place the jar file into the `plugins` directory of your Minecraft server.
3. Restart your server to load the plugin.

## Supported Versions
- Minecraft 1.16.5
- Minecraft 1.17
- Minecraft 1.18

## Migration Notes
- If you are upgrading from a previous version, please ensure to back up your configuration files.
